# 🌋 SKVulcan — Продвинутый ивент «Извержение вулкана»

![Version](https://img.shields.io/badge/Version-1.0--SNAPSHOT-orange)
![API](https://img.shields.io/badge/API-Spigot/Bukkit-gold)
![Dev](https://img.shields.io/badge/Developer-SkyKnock%20DEV-blue)

**SKVulcan** — это мощный плагин на динамический ивент для вашего сервера. Он создает атмосферное извержение вулкана со случайным (или статичным) спавном, BossBar-таймерами и уникальной системой настройки лута прямо в игре.

---

## ✨ Основные фишки

* **🛠 Полная регенерация:** Благодаря WorldEdit, плагин делает бэкап территории и восстанавливает её в исходный вид после ивента.
* **🎮 GUI-Настройка:** Управляйте списком наград, шансами и количеством предметов через удобный интерфейс.
* **🛡 Авто-приват:** WorldGuard автоматически защищает зону вулкана от гриферства на время ивента.
* **🚀 Физика лута:** Предметы эффектно вылетают из жерла вулкана с партиклами и звуками взрывов.
* **📱 Инфо-панель:** BossBar уведомляет всех игроков о текущей стадии (Ожидание, Извержение, Остывание).

---

## 🔧 Командный центр

| Команда | Описание | Доступ |
| :--- | :--- | :--- |
| `/skvulcan` | Открыть GUI настройки лута | `skvulcan.admin` |
| `/skvulcan delay` | Узнать время до ивента или координаты | `skvulcan.admin` |
| `/skvulcan start` | Принудительно запустить вулкан | `skvulcan.admin` |
| `/skvulcan stop` | Мгновенно убрать вулкан и вернуть блоки | `skvulcan.admin` |
| `/skvulcan tp` | Телепорт в эпицентр событий | `skvulcan.admin` |

---

## 📦 Управление лутом в GUI

В меню настройки доступны следующие действия:
* **Drag&Drop:** Просто перетащите предмет из своего инвентаря в меню, чтобы добавить его.
* **ЛКМ / ПКМ:** Изменение шанса выпадения (±1%, с Shift ±10%).
* **Колесико (Middle):** Изменение количества предметов в стаке.
* **Клавиша F:** Настройка "повторов" (сколько раз предмет может выпасть за раз).
* **Клавиша Q (Drop):** Удаление предмета из списка наград.

## ⚙️ Установка

1.  Убедитесь, что у вас установлены **WorldEdit** и **WorldGuard**.
2.  Поместите `SKVulcan.jar` в папку `plugins`.
3.  Загрузите вашу схематику вулкана в `plugins/WorldEdit/schematics/` и укажите название в `config.yml`.
4.  Перезагрузите сервер и настройте лут командой `/skvulcan`.
